# AI Study Buddy

An AI-powered quiz generator that creates 3-question multiple-choice quizzes on any topic.

## Features
- Enter any topic
- Get a 3-question multiple-choice quiz
- Streaming responses
- Mobile responsive

## Tech Stack
- Frontend: HTML + Tailwind CSS
- Backend: FastAPI (Python)
- AI: Groq API (Mixtral-8x7b-32768)
- Deployment: Render.com
